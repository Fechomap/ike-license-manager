# /pm - Orquestacion de Proyecto

Objetivo: **$ARGUMENTS**

Invoca al agente `project-manager` para:
1. Definir alcance y criterios de exito.
2. Construir backlog priorizado.
3. Asignar tareas a agentes especializados.
4. Gestionar riesgos y emitir siguiente accion.
